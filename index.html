Core principle

Accessible, verifiable and reproducible data first.

Every metric should be obtainable, independently checked and capable of being reproduced. Where a metric is unavailable, the framework uses a suitable proxy or excludes it from scoring.

01

Accessible

Use data that can be obtained consistently from public or reliable market sources.

02

Verified

Cross-check critical fields such as spot price, earnings and contract details.

03

Reproducible

Another analyst using the same data should reach a broadly similar conclusion.

04

Transparent

Disclose assumptions, unavailable fields, proxies and model limitations.
